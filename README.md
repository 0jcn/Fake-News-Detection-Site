# Fake News Detection Site
Fake News detection site, being built using React with TypeScript for the frontend, ASP.NET API for connecting to the database and to the Python API for Machine Learning operations. 
# Aims of this project
The main aim of this project is to a create a machine learning model to efficiently and accurately detect fake news, from links being pasted into the site that can then be sent / read from the backend and classified.
# Proposed Features
## Site
- User Accounts
- Ability to give the site links to articles to be classified
- List of articles which have previously been classified, with percentage scores of likeliness to be fake news
## API
- Connect to the database
- Connect to the Python API for machine learning operations
- Act as a middle layer between the site and the db + machine learning model
## Machine Learning
- Be able to classify/ score articles based on likeliness to be fake news
